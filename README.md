- 🔄 Reset / Restart button<br>

- 🎨 Clean UI<br>

## ▶️ How to Play
1. Player X starts the game<br>
2. Players take turns<br>
3. First to get 3 in a row wins<br>
4. Click <b>Reset</b> to play again<br>
